## 🛠️ 技術架構

### 核心檔案結構
```
modules/gui/
├── track_analysis_module.py          # 主模組檔案
├── track_map_widget.py              # 地圖繪製核心
├── track_data_processor.py          # 數據處理器
└── track_analysis_cache.py          # 緩存管理
```

### 關鍵技術要點
- **數據來源**: CLI功能2輸出的JSON檔案
- **繪圖引擎**: matplotlib + PyQt5整合
- **座標系統**: FastF1座標轉換為GUI畫布座標
- **互動功能**: 滑鼠事件處理、縮放平移
